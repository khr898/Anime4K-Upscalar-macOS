# Anime4K Upscaler

A native macOS application for upscaling anime and other animated content using the Anime4K algorithm.

## Description

This application provides a user-friendly graphical interface for the powerful Anime4K video upscaling algorithms. It allows users to easily upscale, compress, and optimize their video files for various purposes, from personal viewing to professional streaming.

The app is built with SwiftUI and is designed to be a modern and efficient tool for video processing on macOS.

## Features

- **Upscaling:** Improve the quality of your videos with a variety of Anime4K modes.
- **Multiple Anime4K Modes:** Choose from a range of Anime4K algorithms to find the best fit for your content.
- **Target Resolutions:** Upscale to 4K, 1440p, 1080p, or 720p.
- **Video Codecs:** Encode your videos using H.264, H.265/HEVC, or AV1.
- **Compression Options:**
    - Use presets for easy configuration.
    - Set a custom quality level (CRF) or a fixed bitrate for more control.
- **Advanced Settings:**
    - Enable Long GOP for smaller file sizes, ideal for streaming.
- **File Management:**
    - Add files via a file dialog or drag-and-drop.
    - View file details, including resolution, size, and duration.
- **Processing:**
    - See the progress of your video processing tasks.
    - Cancel processing at any time.

## How to Use

1.  **Add Files:** Click the "Add Video Files..." button or drag and drop your video files into the application.
2.  **Select a File:** Click on a file in the list to configure its processing options.
3.  **Choose a Mode:** Select an Anime4K mode from the list.
4.  **Set the Resolution:** Choose a target resolution for the output file.
5.  **Select a Codec:** Pick a video codec for the output file.
6.  **Configure Compression:** Use a preset or set a custom quality/bitrate.
7.  **Choose Output Directory:** Select a folder where the processed files will be saved.
8.  **Start Processing:** Click the "Start Processing" button to begin.

## Building from Source

To build the application from source, you will need:

- macOS 14.0 or later
- Xcode 15.0 or later
- Swift 5.9 or later

The project uses a `project.yml` file for project generation. You will also need to have `ffmpeg` installed to bundle it with the application. The `bundle_dependencies.sh` script will handle this for you.

## Dependencies

- **FFmpeg:** This project relies on FFmpeg for video processing. The application includes a script to automatically bundle the `ffmpeg` and `ffprobe` binaries.
